# Login Page - Frontend

Aplicação frontend desenvolvida em **Angular** com foco em autenticação de usuários.

## Objetivo
Este projeto foi criado para praticar a construção de uma interface de login e cadastro integrada com backend.

## Funcionalidades
- Tela de login
- Tela de cadastro
- Validação de campos
- Integração com API backend
- Estrutura pronta para fluxo de autenticação

## Tecnologias utilizadas
- Angular
- TypeScript
- HTML
- CSS

## Como executar
```bash
git clone https://github.com/BrunoAlcantaraGit/login-page-main.git
cd login-page-main
npm install
ng serve
